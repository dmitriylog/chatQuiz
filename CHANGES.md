## 📋 Как использовать

### Аватарки:
1. Откройте профиль (кнопка "Профиль" в шапке)
2. Нажмите "📷 Загрузить" и выберите изображение
3. Для удаления нажмите "🗑️ Удалить"

### Приглашение в квиз:
1. Откройте профиль друга (правый клик на пользователе в списке)
2. Если вы друзья - появится кнопка "🎮 Пригласить в квиз"
3. Друг получит уведомление с вариантами "Принять/Отклонить"
4. При принятии - начнется викторина на двоих

### Адаптивность:
- Приложение автоматически подстраивается под размер окна
- Минимальный размер окна: 800x600
- При изменении размера - элементы пересчитываются

## 🔧 Технические детали

### Адаптивная система (`client/adaptive.py`):
```python
# Примеры использования:
width = self.adaptive.width(100)      # Адаптивная ширина
height = self.adaptive.height(56)     # Адаптивная высота
font_size = self.adaptive.font_size(13)  # Адаптивный шрифт
padding = self.adaptive.padding(12)   # Адаптивный отступ
```

### Аватарки:
- Формат: base64 в WebSocket сообщениях
- Хранение: BLOB in SQLite
- Отображение: круглая маска через PIL
- Fallback: инициалы если PIL не установлен

### При приглашении в квиз:
- Проверяется `room.quiz.active` - не идет ли уже викторина
- Сохраняются `room.quiz.invite_players` - ID игроков
- Оба игрока получают `quiz_invite_accepted` с списком участников
