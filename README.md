# Чат + викторина (футбол)

Пользователи на **разных компьютерах** подключаются к **одному серверу**, общаются в чате и проходят **синхронную викторину** по футболу.

- **SQLite** — история сообщений и вопросы викторины (`data/chat.db`)
- **Викторина** — все видят один вопрос, отвечают A/B/C/D; когда ответили все — следующий вопрос; в конце — таблица и победитель

## Как это работает

```
┌─────────────┐     ┌─────────────┐
│  Компьютер 1│     │  Компьютер 2│
│  python     │     │  python     │
│  main.py    │     │  main.py    │
└──────┬──────┘     └──────┬──────┘
       │    WebSocket       │
       └────────┬───────────┘
                ▼
       ┌─────────────────┐
       │  Сервер чата    │
       │  run_server.py  │
       │  (Yandex Cloud  │
       │   или домашний  │
       │   ПК в сети)    │
       └─────────────────┘
```

1. **Сервер** — один раз запускается на VPS (Yandex Cloud) или на ПК в локальной сети.
2. **Клиент** — на каждом компьютере: `python main.py`, ввод IP сервера и имени.
3. В списке **Участники** видны все онлайн; можно писать в **общий чат** или открыть **личный чат** (клик по имени).

```bash
cd /home/d1m1d1n/chat
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Быстрый тест (два компьютера или два окна)

**Терминал 1 — сервер:**
```bash
python run_server.py --host 0.0.0.0
```

**Терминал 2 и 3 — клиенты:**
```bash
python main.py
```
- Адрес сервера: `127.0.0.1` (на том же ПК) или IP сервера в сети
- Разные имена → увидите друг друга в списке участников

## Yandex Cloud

1. Разверните сервер (Docker/nginx): [deploy/DEPLOY_YANDEX_CLOUD.md](deploy/DEPLOY_YANDEX_CLOUD.md)
2. В клиенте укажите **публичный IP** (порт 80 через nginx) или `:8765` при прямом доступе
3. Включите **WSS**, если используете HTTPS

## Пароль сервера (опционально)

```bash
CHAT_PASSWORD=мой_секрет python run_server.py --host 0.0.0.0
```

Тот же пароль вводится при входе в десктопном клиенте.

## Файлы

| Файл | Назначение |
|------|------------|
| `main.py` | Десктопный клиент |
| `run_server.py` | Сервер для всех пользователей |
| `client/app.py` | Окно приложения |
| `client/connection.py` | Связь с сервером |
| `server.py` | WebSocket API |

Веб-интерфейс в `static/` по-прежнему доступен через браузер, если запущен сервер.

## Развёртывание в облаке

```bash
export CHAT_SERVER=ubuntu@ПУБЛИЧНЫЙ_IP
bash deploy/deploy-from-pc.sh
```

После деплоя в клиенте: адрес `ПУБЛИЧНЫЙ_IP` (порт 80) или `ПУБЛИЧНЫЙ_IP:8765`.
